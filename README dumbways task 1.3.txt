IP Private:
Alamat IP Private digunakan di dalam jaringan lokal (Local Area Network/LAN) dan tidak dapat diakses langsung dari internet. Alamat IP Private diperuntukkan bagi perangkat yang terhubung ke jaringan lokal, seperti komputer, printer, atau router di rumah atau kantor. Beberapa contoh alamat IP Private yang umum digunakan adalah:

192.168.x.x (misalnya, 192.168.0.1)
172.16.x.x hingga 172.31.x.x (misalnya, 172.16.0.1)
10.x.x.x (misalnya, 10.0.0.1)
IP Public:
Alamat IP Public adalah alamat yang digunakan untuk mengidentifikasi perangkat yang terhubung ke internet. Setiap perangkat yang terhubung ke internet memiliki alamat IP Public yang unik. Alamat IP Public memungkinkan perangkat tersebut untuk berkomunikasi dengan perangkat lain di internet. Contoh alamat IP Public adalah 203.0.113.1 atau 2001:0db8:85a3:0000:0000:8a2e:0370:7334 (IPv6).

IP Dynamic:
Alamat IP Dynamic adalah alamat IP yang diberikan secara dinamis kepada perangkat saat terhubung ke jaringan. Saat perangkat terhubung ke jaringan, server DHCP (Dynamic Host Configuration Protocol) di jaringan akan memberikan alamat IP yang tersedia secara dinamis kepada perangkat tersebut. Alamat IP Dynamic dapat berubah setiap kali perangkat terhubung ulang ke jaringan. Keuntungan dari penggunaan IP Dynamic adalah fleksibilitas dan efisiensi penggunaan alamat IP, karena alamat IP dapat digunakan secara bergantian oleh perangkat yang berbeda.

IP Static:
Alamat IP Static adalah alamat IP yang ditetapkan secara manual untuk sebuah perangkat dan tetap tidak berubah seiring waktu. Alamat IP Static digunakan untuk perangkat yang membutuhkan koneksi konstan dan konsisten, seperti server web atau router. Dengan menggunakan alamat IP Static, perangkat tersebut memiliki alamat IP yang tetap dan dapat diakses dengan mudah oleh perangkat lain di jaringan atau internet.
